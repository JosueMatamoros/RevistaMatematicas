Media y Varianza para Binomiales

Si una variable X es Bernoulli entonces la media se puede calcular en forma muy sencilla pues.

 
= 0(1 - p) + 1(p) = p,
mientras que la varianza es:
 
VAR(X) = (0 - p)2(1 - p) + (1 - p)2p = p(1 - p).
