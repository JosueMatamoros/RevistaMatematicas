Si no ve el programa
  • Si hay un mensaje de bloqueo debajo de la barra de direcciones,  debe permitir la ejecución de contenido bloqueado (dé  clic en esta barra)
  • Si no hay mensajes, posiblemente no tenga Java.  Instale Java automáticamente desde internet.

Si tiene este material en cd, instale la máquina virtual de Java ejecutando el archivo

installMVJ-WIN.exe (para windows 98, Mi, etc.)

installMVJ-XP.exe  (para windows XP)

Estos archivos están en la carpeta JAVA del cd.
